# Introduction

### What is FlexiBLE?

FlexiBLE is a toolkit that builds interfaces to embedded systems over Bluetooth Low-Energy and beyond with a focus on data ownership. Sure the cloud is great, and FlexiBLE supports that too, but often proximity and energy efficiency are critical to embedded devices and their data pipelines. FlexiBLE is designed to support high-frequency data collection for the design and research of health wearables but provides generalized functionality for any IoT project. \
